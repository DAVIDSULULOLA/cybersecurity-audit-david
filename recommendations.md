# Recommendations

1. **Install pending security updates immediately** — Two critical security updates (KB5082417 and KB5083769) were found pending. These should be installed as soon as possible to patch known vulnerabilities.

2. **Enable automatic updates** — Configure Windows Update to install updates automatically to prevent future delays in security patching.

3. **Protect browser passwords** — Avoid storing sensitive passwords in Chrome without additional protection. Consider using a dedicated password manager such as Bitwarden or LastPass.

4. **Reduce screen lock timeout** — The device locks after 15 minutes. Consider reducing this to 5 minutes for better physical security.

5. **Enable two-factor authentication** — Turn on 2FA for important accounts such as Google, GitHub, and any financial accounts.

6. **Regularly review installed apps** — Periodically check installed applications and remove any that are unused or no longer needed.

7. **Avoid saving passwords in browsers** — Browser-saved passwords are a single point of failure. A dedicated password manager is safer and more secure.
