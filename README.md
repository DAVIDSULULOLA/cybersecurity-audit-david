# Beginner Cybersecurity Audit Project

## Name
David 

## Project Title
Personal Laptop Cybersecurity Audit

## Objective
The aim of this project is to identify basic cybersecurity weaknesses in a personal laptop and suggest practical security improvements.

## Scope
I audited my personal laptop running Windows 11.

## Tools Used
- Windows Security (Windows Defender)
- Windows Defender Firewall
- Windows Update
- Sign-in options (Accounts settings)
- Installed Apps settings
- Chrome Password Manager

## Files in this Repository
- checklist.md
- findings-report.md
- recommendations.md
- screenshots/

## Key Findings
- Antivirus and Firewall were active and properly configured
- Two critical security updates were pending at the time of audit
- Screen lock and PIN protection were enabled
- No suspicious applications were found
- Browser passwords were stored without a master password

## Conclusion
This audit revealed that the laptop has a solid basic security foundation, but requires immediate attention to pending OS updates and browser password protection to reduce overall risk.
