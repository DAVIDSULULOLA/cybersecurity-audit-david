# Findings Report

## Target Audited
Personal Laptop (Windows 11)

## Summary
I carried out a basic cybersecurity audit on my personal laptop to identify common security weaknesses and assess the overall security posture of the device.

## Findings

1. **Antivirus is active** — Windows Defender is installed and running with no threats detected.
2. **Firewall is enabled** — Windows Defender Firewall is on and blocking unauthorized incoming connections.
3. **Operating system is not fully updated** — Two security updates were found pending at the time of audit (KB5082417 and KB5083769).
4. **Screen lock is enabled** — The device automatically locks after 15 minutes of inactivity.
5. **PIN protection is set** — Windows Hello PIN is configured and required to sign in.
6. **No suspicious apps found** — All installed applications were recognized and legitimate.
7. **Browser passwords are unprotected** — Chrome stores saved passwords without a master password, meaning anyone with access to the browser can view them.

## Risk Level
Medium Risk

## Why?
The device has strong basic protections in place. However, pending security updates and unprotected browser passwords are vulnerabilities that could be exploited if not addressed.
